== Description ==

A simple one word language translator that uses machine translation.  Developed at Translation Services USA using the Microsoft Bing machine translation API, formerly using the google API until they started charing for it.  This plugin is provided free of charge.  Translating more than one word is possible with this plugin although not recomended due to the size of the of the input and result text area boxes.

More info at http://www.translation-services-usa.com/owt-download.php

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `owt-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place owt-plugin in your widget section in your wordpress dashboard, or copy iFrame code into a post.

== Frequently Asked Questions ==

= What is the maximun number of words that can be translated at one time? =

Technically at the moment the One Word Translator can translate and unlimted number of words but it is recomended that you do one at a time.  Limiting the amount of words may be implemented in a later update.

= Can I get the OWT in another color? =

Hot pink is the only one available for now, but option for a blue and orange background will be available in a future update.

= Can I get the iframe code directly? =

Yes, go to http://www.translation-services-usa.com/owt-download.php
